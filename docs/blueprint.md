# **App Name**: NutriScan

## Core Features:

- Menu Scanner: Use AI to scan the restaurant menu photo for text, and interpret each menu item into food options that the user can confirm. The AI tool can deal with language translation automatically.
- Nutritional Data Generation: Generate nutritional information for each item based on ingredients using a nutritional database API (using an AI tool to augment/improve information and formatting when possible).
- Augmented Menu Display: Display the original menu photo with tappable overlays indicating the nutritional information for each item.
- Detailed Nutrition View: Detailed view showing the full nutritional breakdown, ingredients, and allergen information for a menu item, upon user tap.

## Style Guidelines:

- Primary color: Soft Green (#A7D1AB), evoking a sense of health and nature.
- Background color: Light Beige (#F5F5DC), providing a neutral and clean backdrop.
- Accent color: Muted Orange (#E5B775), used for interactive elements and highlights to draw attention without overwhelming the user.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look; suitable for both headlines and body text.
- Use clean and simple icons to represent different nutritional values and allergen information.
- Maintain a clean and intuitive layout to ensure ease of use.
- Implement subtle animations and transitions for a smooth user experience when viewing nutritional information.
